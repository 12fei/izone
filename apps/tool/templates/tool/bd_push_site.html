工具说明：

  • 这个Sitemap主动推送工具是百度主动推送工具的升级版
  • 本工具的作用是提取网站sitemap页面中所有的链接，并且使用百度站长官方提供的API主动推送链接给百度收录，百度主动推送的作用请查看 官方说明
  • 首先将百度站长提供的“接口调用地址”复制到下面表单中，然后将自己网站的sitemap页面的地址复制到对应表单中，点击提交按钮即可看到返回的信息
  • 想了解这个工具的原理和查看源代码，可以前往 百度主动推送工具，有任何问题都可以到文章底部评论留言
注：请填写百度站长平台提供给您的接口调用地址
注：提交的链接的域名必须与接口中site保持一直才能推送成功
推送结果

推送结果参数解读：

  • 推送成功
  • 推送失败

状态码为200，可能返回以下字段：

字段 是否必传 参数类型 说明
remain 是 int 当天剩余的可推送url条数
success 是 int 成功推送的url条数
not_same_site 否 array 由于不是本站url而未处理的url列表
not_valid 否 array 不合法的url列表

成功返回示例：

{
    "remain":4999998,
    "success":2,
    "not_same_site":[],
    "not_valid":[]
}

状态码为4xx，返回字段有：

字段 是否必传 参数类型 说明
error 是 int 错误码，与状态码相同
message 是 string 错误描述

失败返回示例：

{
    "error":401,
    "message":"token is not valid"
}
